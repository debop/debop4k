# debop4k-timeperiod

joda-time의 DateTime 을 이용하여 일자, 기간등의 다양한 자료 형식과 연산을 제공합니다.

### Change Logs


1.0.0 : 

	DateTime 관련 Helper Methods 제공
	기간에 대한 여러가지 자료구조 제공 (TimePerid, TimeRange, TimeBlock 등)
	기간들에 대한 관계 (Intersection, Overlaps 등 Relation을 판단)
	년/월/주/일/시/분 등 시간 단위별 기간에 대한 기본 클래스 제공 (YearRange, MonthRange, WeekRange, DayRange ...)
	특정한 Calendar에 대한 기간 산정 (휴일 제외, Working Time 기준 등에 따른 기간 산정)
