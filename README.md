# debop4k [![Build Status](https://travis-ci.org/debop/debop4k.png)](https://travis-ci.org/debop/debop4k)

Backend library for Kotlin JVM

## Modules

  * Config
  * Core
  * Data
  * Data for MyBatis
  * Data for ORM
  * Data for MongoDB
  * Data for Redis
  * Data for Apache Ignite
  * Timeperiod

## Build from Source

Maven
 
    ./mvnw clean install
